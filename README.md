# Heart-Disease-Prediction
Machine Learning project for heart disease prediction with data preprocessing, EDA, SQL, Jupyter Notebook, and Power BI dashboard.

# ❤️ Heart Disease Prediction
## 📌 Overview
This project analyzes heart disease data using **Python, SQL, Jupyter Notebook, Machine Learning, and Power BI**. It includes data cleaning, exploratory data analysis (EDA), predictive modeling, and an interactive dashboard to generate healthcare insights.

## 🚀 Features
- Data Cleaning & Preprocessing
- SQL Database Queries
- Exploratory Data Analysis (EDA)
- Machine Learning Prediction
- Model Evaluation
- Interactive Power BI Dashboard

## 🛠️ Tech Stack
- Python
- Jupyter Notebook
- SQL
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Power BI
- Git & GitHub

## 📊 Workflow
Data Collection → Data Cleaning → SQL → EDA → Machine Learning → Model Evaluation → Power BI Dashboard

## 🎯 Objective
To predict heart disease risk using machine learning and visualize key health insights through an interactive dashboard.
